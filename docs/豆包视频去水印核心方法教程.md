# 豆包视频去水印核心方法教程

## 一、核心原理

豆包视频有**两套 API**，返回不同版本的视频：

### 1. 带水印版本（旧 API）

```
POST https://www.doubao.com/creativity/share/get_video_share_info
```

**返回的视频 URL 特征**：
- 域名：`v*-default.365yg.com`
- 参数：`lr=video_gen_watermark_dyn`（动态水印标识）
- 路径：`oQDCr4YI1BetAWbK0QkFwh1Erg4UAI3ysEsAQy/`
- 结果：视频有水印

### 2. 无水印版本（新 API）⭐

```
POST https://www.doubao.com/samantha/media/get_play_info
```

**返回的视频 URL 特征**：
- 域名：`v*-videoweb.doubao.com`
- 参数：`cr=0&dr=0&er=0&download=true`（无水印标识）
- 路径：`oYYA4Kkb0slhEErDCQ1QeAwFIK4tIyU1IANBsU/`
- 结果：**视频无水印**

## 二、关键发现

### 为什么之前的去水印方法都失败了？

很多人尝试：
- ❌ 移除 URL 参数 `lr=video_gen_watermark_dyn` → 返回 403 错误
- ❌ 替换参数为 `lr=none` → 返回 403 错误
- ❌ 使用视频处理工具（FFmpeg、OpenCV）→ 画质损失，效果差

**根本原因**：水印是服务器端动态生成的，URL 参数只是标识，无法通过修改参数去除。

### 正确方法

使用官方的 **无水印 API**，直接获取原始视频文件。

### 步骤 1：提取 video_id

从分享链接中提取 `video_id` 参数。

**示例链接**：
```

```

**提取方法**：
```python

url = "https://www.doubao.com/video-sharing?share_id=xxx&video_id=v0269cg10004d82qodiljht41c5h83qg"
parsed = urlparse(url)
params = parse_qs(parsed.query)
video_id = params['video_id'][0]  # v0269cg10004d82qodiljht41c5h83qg
```

**请求**：
```http

```

## 四、完整代码示例

### Python 完整示例

```python
#!/usr/bin/env python3
"""
豆包视频无水印下载 - 完整示例
"""

import requests
from urllib.parse import urlparse, parse_qs


def download_doubao_video(share_url, output_path="output.mp4"):
    """
    下载豆包无水印视频
    
    Args:
        share_url: 豆包视频分享链接
        output_path: 输出文件路径
    """
    
    # 1. 提取 video_id
    parsed = urlparse(share_url)
    params = parse_qs(parsed.query)
    video_id = params.get('video_id', [None])[0]
    
    if not video_id:
        raise ValueError("无法从 URL 中提取 video_id")
    
    print(f"视频 ID: {video_id}")
    
    # 2. 调用无水印 API
    api_url = "https://www.doubao.com/samantha/media/get_play_info"
    headers = {
        'Content-Type': 'application/json',
        'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36',
        'Referer': 'https://www.doubao.com/',
        'Accept': 'application/json'
    }
    data = {"key": video_id}
    
    print("正在获取视频信息...")
    response = requests.post(api_url, headers=headers, json=data)
    result = response.json()
    
    if result.get('code') != 0:
        raise Exception(f"API 调用失败: {result.get('msg')}")
    
    # 3. 获取无水印视频 URL（优先 1080p）
    if result['data'].get('original_media_info'):
        video_url = result['data']['original_media_info']['main_url']
        meta = result['data']['original_media_info']['meta']
        quality = "1080p"
    else:
        video_url = result['data']['media_info'][0]['main_url']
        meta = result['data']['media_info'][0]['meta']
        quality = "720p"
    
    print(f"视频质量: {quality}")
    print(f"分辨率: {meta['width']}x{meta['height']}")
    print(f"时长: {meta['duration']}秒")
    
    # 4. 下载视频
    print("正在下载视频...")
    video_response = requests.get(
        video_url,
        headers={'Referer': 'https://www.doubao.com/'},
        stream=True
    )
    
    total_size = int(video_response.headers.get('content-length', 0))
    downloaded = 0
    
    with open(output_path, 'wb') as f:
        for chunk in video_response.iter_content(chunk_size=8192):
            if chunk:
                f.write(chunk)
                downloaded += len(chunk)
                if total_size > 0:
                    progress = (downloaded / total_size) * 100
                    print(f"\r下载进度: {progress:.1f}%", end='')
    
    print(f"\n✅ 下载完成: {output_path}")
    print(f"文件大小: {downloaded / 1024 / 1024:.2f} MB")


# 使用示例
if __name__ == '__main__':
    share_url = "https://www.doubao.com/video-sharing?share_id=44962654004563970&video_id=v0269cg10004d82qodiljht41c5h83qg"
    download_doubao_video(share_url, "doubao_video_no_watermark.mp4")
```

### JavaScript/Node.js 示例

```javascript
const https = require('https');
const fs = require('fs');
const url = require('url');

async function downloadDoubaoVideo(shareUrl, outputPath) {
  // 1. 提取 video_id
  const parsedUrl = new URL(shareUrl);
  const videoId = parsedUrl.searchParams.get('video_id');
  
  if (!videoId) {
    throw new Error('无法从 URL 中提取 video_id');
  }
  
  console.log(`视频 ID: ${videoId}`);
  
  // 2. 调用无水印 API
  const postData = JSON.stringify({ key: videoId });
  
  const options = {
    hostname: 'www.doubao.com',
    port: 443,
    path: '/samantha/media/get_play_info',
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36',
      'Referer': 'https://www.doubao.com/',
      'Accept': 'application/json',
      'Content-Length': Buffer.byteLength(postData)
    }
  };
  
  console.log('正在获取视频信息...');
  
  return new Promise((resolve, reject) => {
    const req = https.request(options, (res) => {
      let data = '';
      
      res.on('data', (chunk) => {
        data += chunk;
      });
      
      res.on('end', () => {
        try {
          const result = JSON.parse(data);
          
          if (result.code !== 0) {
            reject(new Error(`API 调用失败: ${result.msg}`));
            return;
          }
          
          // 3. 获取无水印视频 URL
          const videoUrl = result.data.original_media_info?.main_url || 
                          result.data.media_info[0].main_url;
          const meta = result.data.original_media_info?.meta || 
                      result.data.media_info[0].meta;
          
          console.log(`分辨率: ${meta.width}x${meta.height}`);
          console.log(`时长: ${meta.duration}秒`);
          
          // 4. 下载视频
          console.log('正在下载视频...');
          downloadFile(videoUrl, outputPath)
            .then(() => {
              console.log(`✅ 下载完成: ${outputPath}`);
              resolve();
            })
            .catch(reject);
          
        } catch (e) {
          reject(e);
        }
      });
    });
    
    req.on('error', reject);
    req.write(postData);
    req.end();
  });
}

function downloadFile(url, outputPath) {
  return new Promise((resolve, reject) => {
    const file = fs.createWriteStream(outputPath);
    
    https.get(url, { headers: { 'Referer': 'https://www.doubao.com/' } }, (response) => {
      response.pipe(file);
      file.on('finish', () => {
        file.close();
        resolve();
      });
    }).on('error', (err) => {
      fs.unlink(outputPath, () => {});
      reject(err);
    });
  });
}

downloadDoubaoVideo(shareUrl, 'output.mp4');
```

## 五、关键技术点

### 1. API 端点对比

| 特性 | 带水印 API | 无水印 API ⭐ |
|------|-----------|--------------|
| 端点 | `/creativity/share/get_video_share_info` | `/samantha/media/get_play_info` |
| 参数 | `share_id`, `vid` | `key` (video_id) |
| 返回域名 | `v*-default.365yg.com` | `v*-videoweb.doubao.com` |
| URL 参数 | `lr=video_gen_watermark_dyn` | `cr=0&dr=0&er=0&download=true` |
| 水印 | ✅ 有水印 | ❌ 无水印 |

### 2. 为什么必须在浏览器环境中调用？

API 需要验证：
- Cookie（用户身份）
- Referer（来源验证）
- User-Agent（客户端识别）

**解决方案**：
1. 使用浏览器自动化工具（Puppeteer、Playwright）
2. 在浏览器控制台中直接调用
3. 使用抓包工具获取 Cookie

### 3. 浏览器控制台调用方法

打开豆包视频分享页面，按 F12 打开控制台，执行：


## 六、常见问题

### Q1: API 返回 403 错误？

**原因**：缺少必要的请求头或 Cookie

**解决方案**：
```python
# 添加完整的请求头
headers = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36',
    'Referer': 'https://www.doubao.com/',
    'Accept': 'application/json, text/plain, */*',
    'Accept-Language': 'zh-CN,zh;q=0.9',
    'Origin': 'https://www.doubao.com'
}

# 或者在浏览器环境中调用
```

### Q2: 视频下载后还是有水印？

**原因**：使用了错误的 API 端点

**检查方法**：
```python
# 检查 URL 域名
if '365yg.com' in video_url:
    print("❌ 这是带水印的 URL")
elif 'videoweb.doubao.com' in video_url:
    print("✅ 这是无水印的 URL")
```

### Q3: 如何获取更高画质的视频？

**答案**：使用 `original_media_info` 字段

```python
# 优先使用原始版本
if result['data'].get('original_media_info'):
    video_url = result['data']['original_media_info']['main_url']
    # 这是最高画质的版本
```

### Q4: URL 有时效性吗？

**答案**：是的，URL 会过期

**建议**：获取 URL 后立即下载，不要存储 URL
