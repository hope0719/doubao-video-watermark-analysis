# 豆包视频去水印 - 快速参考卡片

## 核心方法（一句话）

**使用 `/samantha/media/get_play_info` API，传入 `video_id`，获取无水印视频 URL。**

---

## API 对比

| 项目 | 带水印 API ❌ | 无水印 API ✅ |
|------|--------------|--------------|
| **端点** | `/creativity/share/get_video_share_info` | `/samantha/media/get_play_info` |
| **参数** | `share_id`, `vid` | `key` |
| **域名** | `v*-default.365yg.com` | `v*-videoweb.doubao.com` |
| **URL 参数** | `lr=video_gen_watermark_dyn` | `cr=0&dr=0&er=0&download=true` |
| **结果** | 有水印 | **无水印** |

---

## 三步搞定

### 步骤 1：提取 video_id

```python
from urllib.parse import urlparse, parse_qs

url = "https://www.doubao.com/video-sharing?share_id=xxx&video_id=v0269cg10004d82qodiljht41c5h83qg"
video_id = parse_qs(urlparse(url).query)['video_id'][0]
```

### 步骤 2：调用 API

```python
import requests

response = requests.post(
    "https://www.doubao.com/samantha/media/get_play_info",
    headers={
        'Content-Type': 'application/json',
        'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36',
        'Referer': 'https://www.doubao.com/'
    },
    json={"key": video_id}
)
result = response.json()
```

### 步骤 3：获取 URL

```python
# 1080p 原始无水印（推荐）
video_url = result['data']['original_media_info']['main_url']

# 或 720p 无水印
video_url = result['data']['media_info'][0]['main_url']
```

## 完整代码（复制即用）

```python
#!/usr/bin/env python3
import requests
from urllib.parse import urlparse, parse_qs

def download_doubao_video(share_url, output="output.mp4"):
    # 提取 video_id
    video_id = parse_qs(urlparse(share_url).query)['video_id'][0]
    
    # 调用 API
    res = requests.post(
        "https://www.doubao.com/samantha/media/get_play_info",
        headers={
            'Content-Type': 'application/json',
            'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36',
            'Referer': 'https://www.doubao.com/'
        },
        json={"key": video_id}
    )
    
    # 获取无水印 URL
    data = res.json()['data']
    video_url = data.get('original_media_info', {}).get('main_url') or data['media_info'][0]['main_url']
    
    # 下载视频
    video_res = requests.get(video_url, headers={'Referer': 'https://www.doubao.com/'})
    with open(output, 'wb') as f:
        f.write(video_res.content)
    
    print(f"✅ 下载完成: {output}")

# 使用
download_doubao_video(
    "https://www.doubao.com/video-sharing?share_id=xxx&video_id=xxx",
    "video.mp4"
)
```

## 浏览器控制台快速测试

打开视频页面 → F12 → 控制台 → 粘贴：

```javascript
const videoId = new URLSearchParams(location.search).get('video_id');
fetch('https://www.doubao.com/samantha/media/get_play_info', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({key: videoId})
})
.then(r => r.json())
.then(d => {
  console.log('720p:', d.data.media_info[0].main_url);
  console.log('1080p:', d.data.original_media_info.main_url);
});
```

---

## 注意事项

1. **必须在浏览器环境**或使用正确的 Cookie
2. **URL 会过期**，获取后立即下载
3. **优先使用 1080p**：`original_media_info.main_url`
4. **检查域名**：`videoweb.doubao.com` 才是无水印
