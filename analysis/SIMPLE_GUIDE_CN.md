# 🌱 超简单数据挖掘指南

## 🎯 专为新手设计的3种方法

### 🌟 方法1: 浏览器F12工具 (最简单)

#### 📱 **只需6步**:

**步骤1: 打开豆包分享链接**
```
1. 在Chrome浏览器中打开您的豆包分享链接
2. 确保视频页面正常加载
```

**步骤2: 打开开发者工具**
```
1. 按键盘上的 F12 键
   (或右键页面 → 选择"检查")
2. 会出现一个黑色窗口
```

**步骤3: 找到Network标签**
```
1. 在黑色窗口顶部，点击"Network"标签
2. 确保左下角的录制按钮是红色的(在录制)
```

**步骤4: 播放视频**
```
1. 回到页面，点击播放视频
2. 等待视频开始播放
3. 黑色窗口中会出现很多条目
```

**步骤5: 找到视频文件**
```
在Network列表中:
- 寻找包含"video"、"mp4"、"stream"的行
- 寻找文件大小很大的行(几百KB到几MB)
- 点击这些行，查看右侧详情
```

**步骤6: 保存数据**
```
1. 右键找到的视频请求
2. 选择"Copy" → "Copy response"
3. 粘贴到记事本中保存
```

#### 📝 **查找技巧**:
- 🔍 **按类型筛选**: 在Network面板顶部，点击"Media"
- 🎯 **搜索功能**: 按Ctrl+F，搜索"mp4"、"video"、"cdn"
- 💾 **保存文件**: 右键 → "Save as HAR with content"

### 🌟 方法2: 浏览器插件 (一键检测)

#### 🚀 **推荐插件**:

**Video DownloadHelper**
```
安装步骤:
1. 打开Chrome应用商店
2. 搜索"Video DownloadHelper"
3. 点击"添加到Chrome"
4. 安装完成后刷新豆包页面
```

**使用方法**:
```
1. 打开豆包分享链接
2. 播放视频
3. 点击浏览器右上角插件图标
4. 插件会自动检测所有视频源
5. 点击检测到的视频可以查看详细信息
```

#### 🎁 **其他有用插件**:
- **Web Developer**: 查看页面所有资源
- **JSON Viewer**: 自动格式化JSON数据
- **EditThisCookie**: 查看和管理网站Cookie

### 🌟 方法3: 在线工具 (零安装)

#### 🌐 **推荐在线网站**:

**网站1: Web Sniffer**
- 网址: https://web-sniffer.net/
- 使用方法: 
  ```
  1. 打开网站
  2. 在输入框粘贴豆包链接
  3. 点击Submit
  4. 查看完整响应内容
  ```

**网站2: HTTP Status Checker**
- 网址: https://httpstatus.io/
- 用途: 检查链接状态和重定向

**网站3: Base64 Decode Online** 
- 网址: https://www.base64decode.org/
- 用途: 如果找到Base64编码的数据，这里可以解码

#### 📋 **在线工具使用流程**:
```
1. ⭐ 用Web Sniffer获取完整响应
2. ⭐ 复制找到的疑似编码内容
3. ⭐ 用在线解码工具尝试解码
4. ⭐ 保存成功的结果
```
